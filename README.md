# Основная информация
Репозиторий содержит исходные коды чата, позволяющего пользователям на разных операционных системах общаться друг с другом.

# Установка репозитория
Вы можете установить репозиторий двумя способами:
* Скачать zip-архив репозитория.
* Склонировать репозиторий.

Внешние зависимости репозитория организованы как подмодули github. Из-за этого, если вы решили склонировать репозиторий, то для полной установки репозитория вам необходимо запустить `afterDownload.bat` или `afterDownload.sh` или выполнить следующие в консоли следующие команды:
```bash
cd CrossplatformChat
git submodule update --init --recursive
```

# Автоматическая сборка проекта
Для автоматической сборки проекта запустите файл `build.bat` или `build.sh`.

# Сборка проекта вручную
Для сборки проекта вам понадобится `cmake`. Скачать его можно с [*официального сайта*](https://cmake.org/download/). После установки cmake выполните следующую последовательность действий в терминале или консоли:
```bash
cd CrossplatformChat
mkdir build
cd build
cmake ../src
cmake --build .
```
После этого в директории `build/bin` будет лежать нужный исполняемый файл.

# Запуск сервера в автоматическом режиме
Для запуска сервера в автоматическом режиме запустите файл `startServer.bat` или `startServer.sh`.

# Запуск клиента в автоматическом режиме
Для запуска клиента в автоматическом режиме запустите файл `startClient.bat` или `startClient.sh`.

# Запуск вручную
Для запуска приложения перейдите в папку `build/bin`, которая была создана после сборки и выполните следующие команды:
* **`./CrosspatformChat.exe --server`** -- если вы хотите запустить программу как сервер.
* **`./CrosspatformChat.exe --client`** -- если вы хотите запустить программу как клиент.
